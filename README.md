# Kaf's Valentine Special

Kaf's Valentine Special adds themed cookies, cotton candy, Aristea flowers, and the Lovey Dovey Infuser.

## Support

- Minecraft `26.1.2`
- Fabric
- NeoForge

## Dependencies

- [Amber](https://modrinth.com/mod/amber)
- Fabric API on Fabric

## Highlights

- 20 cookie variants with upgraded versions
- Cotton Candy crops and food items
- The Lovey Dovey Infuser
- Aristea world generation
